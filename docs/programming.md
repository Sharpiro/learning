# programming

## bitwise ops

- and
  - `0x2bffff & 0xff12ff`
    - returns `0x2b12ff`
  - `x & 255`
    - returns `x`
  - `x & 0`
    - returns all 0's
- or
  - used for merging 2 numbers
  - `0x001200 | 0xff00ff`
    - returns `0xff12ff`
  - `x | 255`
    - returns all 1's
  - `x | 0`
    - returns 'x'
